# VHDL Digital Design

This repository contains basic digital logic designs implemented in VHDL, along with testbenches for simulation and verification.

## Implemented

- OR Gate
- AND Gate
- XOR Gate

## Structure

Each module includes:
- A design file (.vhd)
- A testbench with assertions

## Tools

- VHDL (IEEE STD_LOGIC_1164)
- VS Code
- EDA Playground (Riviera-PRO EDU)

## How to Run

1. Open https://www.edaplayground.com/
2. Paste design code into the Design panel
3. Paste testbench into the Testbench panel
4. Set top entity to testbench
5. Click Run

## Notes

- Testbenches use assertions to automatically verify correctness
- This project is for learning and practicing digital logic design
